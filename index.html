Module 1: Python Basics

Introduction to Python Programming

Python is a high-level, interpreted programming language known for its readability and versatility. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages in the world.

In this module, you will learn the fundamental building blocks of Python programming, including variables, data types, basic operators, and how to write your first Python script.

Python uses indentation to define code blocks, making the code visually clean and easy to read. Unlike many other programming languages, Python does not use curly braces or semicolons to delimit statements.

# Your first Python program name = "Sally" print("Hello, " + name + "! Welcome to Python.")

The example above demonstrates variable assignment and string concatenation, two concepts you will master in this module.

Increased Difficulty · Data Structures

Dictionary Iteration

smart_toy AI Coaching Prompt

Write a Python loop that iterates over the following dictionary and prints each key-value pair in the format key: value.

student_scores = {"Alice": 92, "Bob": 87, "Charlie": 95}

Sally's Answer

for key in student_scores: print(student_scores)
error Incorrect — Here's what happened

Your loop iterates over the dictionary keys correctly, but print(student_scores) prints the entire dictionary on each iteration instead of the individual key-value pair.

To access both the key and value, use the .items() method and unpack into two variables:

for key, value in student_scores.items(): print(f"{key}: {value}")

Module 1: Python Basics

Introduction to Python Programming

Python is a high-level, interpreted programming language known for its readability and versatility. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages in the world.

In this module, you will learn the fundamental building blocks of Python programming, including variables, data types, basic operators, and how to write your first Python script.

Python uses indentation to define code blocks, making the code visually clean and easy to read. Unlike many other programming languages, Python does not use curly braces or semicolons to delimit statements.

# Your first Python program name = "Sally" print("Hello, " + name + "! Welcome to Python.")

The example above demonstrates variable assignment and string concatenation, two concepts you will master in this module.

Python is a high-level, interpreted programming language known for its readability and versatility. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages in the world.

In this module, you will learn the fundamental building blocks of Python programming, including variables, data types, basic operators, and how to write your first Python script.

Python uses indentation to define code blocks, making the code visually clean and easy to read. Unlike many other programming languages, Python does not use curly braces or semicolons to delimit statements.

# Your first Python program name = "Sally" print("Hello, " + name + "! Welcome to Python.")

The example above demonstrates variable assignment and string concatenation, two concepts you will master in this module.
