Dictionary Iteration

smart_toy AI Coaching Prompt
Write a Python loop that iterates over the following dictionary and prints each key-value pair in the format key: value.
student_scores = {"Alice": 92, "Bob": 87, "Charlie": 95}

Sally's Answer

for key in student_scores: print(student_scores)
cancel Incorrect — Here's what happened
Your loop iterates over the dictionary keys correctly, but print(student_scores) prints the entire dictionary on each iteration instead of the individual key-value pair.

To access both the key and value, use the .items() method and unpack into two variables:

for key, value in student_scores.items(): print(f"{key}: {value}")
